Medical Record System (Курсова робота)
Тема: Програмне забезпечення для ведення базової електронної медичної карти пацієнта.
Опис
Цей проєкт реалізує систему управління медичними записами. Система побудована за клієнт-серверною архітектурою:
•	Backend (Ядро): Реалізовано на C++ (OOP, класи Doctor, Patient, Appointment, Prescription).
•	Frontend (Інтерфейс): Веб-застосунок на Python (Flask).
Структура проєкту
•	backend/ - Вихідний код C++ та CMake конфігурація.
•	frontend/ - Веб-сервер на Python та HTML шаблони.
Вимоги для запуску
•	CMake
•	C++ компілятор (Clang/GCC)
•	Python 3 + Flask
